# Personal Portfolio Website

This repository contains the source code for my personal software developer portfolio.

🌐 **Live Site:**  
https://antonioc-26.github.io

The portfolio showcases my full-stack, data-driven, and client-based software projects, along with my technical skills and contact information for recruiters and collaborators.

---

## Overview

This portfolio is designed to present my strongest work in a clean, recruiter-friendly format. It highlights real-world projects, interactive applications, and data-focused systems, with an emphasis on usability, performance, and modern development practices.

---

## Features

- Responsive single-page portfolio layout
- Circular profile image for personal branding
- Featured project showcase with screenshots
- Clickable project previews linking to live demos
- GitHub repository links for each project
- Resume **view and download** functionality
- Scroll-based navigation highlighting
- Smooth scrolling between sections
- Interactive UI elements and hover effects
- Contact section with direct links to email, GitHub, and LinkedIn

## Featured Projects

The portfolio highlights several projects that demonstrate different areas of software engineering:

### 🥇 Millersville Tennis Club Website
- Real client project built with React and Vite
- Focused on responsive design and modern UI/UX
- Deployed using Vercel

### 🛒 X-Industries (Amazon 2.0)
- Full-stack e-commerce application
- Built with Node.js, Express, and MongoDB
- Includes authentication, cart system, and order workflow

### 🧠 Maverick Course Scheduler
- Constraint-based scheduling system using Python and Z3
- Includes optimization logic and schedule generation
- Integrated with a Flask-based interface

### 📊 CS Job Market Dashboard
- Data analytics dashboard analyzing entry-level CS job postings
- Built with Python, Pandas, Vue, and Chart.js
- Features interactive filtering and visualization

### 📌 Job Tracker
- Lightweight CRUD application for tracking job applications
- Includes filtering, dashboard stats, and localStorage persistence

### 🎮 Web Games Collection
- Interactive browser-based games built with JavaScript
- Includes RPSLS, Memory Game, and Nine Men’s Morris
- Focuses on UI interactivity and game logic

## Running Locally

Clone the repository:

```bash
git clone https://github.com/antonioc-26/antonioc-26.github.io.git
cd antonioc-26.github.io
```

Then open:
```
index.html
```
in your browser, or use a local development server such as VS Code Live Server.
