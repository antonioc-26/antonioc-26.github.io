# Personal Portfolio Website

This repository contains the source code for my personal developer portfolio.

Live site:
https://antonioc-26.github.io

The portfolio highlights my software development projects, technical skills, and contact information for recruiters and collaborators.

---

## Features

- Responsive single-page portfolio layout
- Project showcase with screenshots
- GitHub repository links
- Live demo links for deployed projects
- Resume download
- Contact information
- Scroll-activated navigation highlighting
- Interactive UI elements and hover effects

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages

---

## Featured Projects

The portfolio highlights several projects including:

- Maverick Course Scheduler
- X-Industries (Amazon-style full-stack application)
- Web Games Collection
- Job Tracker

Each project includes screenshots, descriptions, and links to both the source code and live demos where available.
